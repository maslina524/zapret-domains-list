# zapret-domains-list

Домены будут добавляться вручную, по мере нахождения

## Как использовать

Скачайте файл и замените его в `.../zapret-discord-youtube-x.x.x/lists/`

Псоле установки зайдите в zapret (service.bat) и переустановите тот же конфиг который у вас установлен, это полностью перезапустит zapret и он будет работать с новым списком доменов

## Быстрое скачивание

**Windows:**
```powershell
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/maslina524/zapret-domains-list/refs/heads/main/list-general.txt" -OutFile "list-general.txt"
```

**Для всех платформ:**
```cmd
curl -o list-general.txt "https://raw.githubusercontent.com/maslina524/zapret-domains-list/refs/heads/main/list-general.txt"
```
